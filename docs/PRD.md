# PRD for TerrorGym (HIIT Class Scheduling and Reservations)

## Overview

This document outlines the Product Requirements Document (PRD) for the TerrorGym's HIIT class scheduling and reservations system.

### Key Features

- **Payments**: No payments required.
- **Waitlist**: Yes, a waitlist will be available for full classes.
- **Notifications**: Notifications are not included in the system.
- **Cancellations**: Users can cancel their reservations until 45 minutes before class starts.

### Schedule

Classes are scheduled in fixed weekly time slots:
- **Monday and Friday**: 09:30 - 10:15
- **Tuesday, Wednesday, and Thursday**:
  - 19:00 - 19:45
  - 20:00 - 20:45
  - 21:00 - 21:45

### Administration

- **Admin**: There will be a single administrator managing the system.
- **Location**: Classes will be held at a single location.
