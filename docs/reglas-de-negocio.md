# Reglas de negocio

## Horarios fijos
- Lunes y viernes: 1 clase de 09:30 a 10:15.
- Martes, miercoles y jueves: 3 clases 19:00-19:45, 20:00-20:45, 21:00-21:45.
- El horario puede variar puntualmente (admin puede modificar sesiones).

## Aforo
- Maximo 15 alumnos por sesion.
- Cuando el aforo se completa, las nuevas solicitudes van a lista de espera.

## Bonos
- Bono 2 clases semanales: 20 EUR.
- Bono 3 clases semanales: 30 EUR.
- Los importes pueden cambiar en el futuro.
- El admin registra manualmente el pago y el tipo de bono.

## Reservas semanales
- Las reservas se realizan por semana.
- El alumno solo puede reservar dentro de los limites de su bono.

## Cancelacion y penalizacion
- Cancelacion o cambio permitido hasta 45 minutos antes del inicio.
- Si la cancelacion se hace dentro de 45 minutos, se consume la sesion como realizada.

## Lista de espera
- Si se libera cupo, se avanza la lista de espera por orden de ingreso.
- Notificacion basica pendiente de definicion (placeholder).

## Asistencia
- La asistencia se confirma al inicio de la sesion.
- Si el alumno no asiste y no cancelo a tiempo, la sesion se consume.

## Alta de usuarios
- Solo el admin puede crear usuarios.
- El admin registra email y/o telefono.
- Se envia enlace o mensaje para establecer PIN.
